AniTrack — a single-page anime watchlist. Vanilla HTML, CSS, and JavaScript only; no frameworks, no build step. Features: add an anime with a title and total episode count; increase or decrease episodes watched with buttons; an entry automatically becomes Completed when episodes watched equals total episodes; rate a completed anime 1 to 5 stars; filter tabs for All, Watching, and Completed; delete an entry; all data persists in localStorage across reloads; show an empty-state message when the list is empty. Dark theme, responsive on mobile. Include a README with setup and usage instructions.
